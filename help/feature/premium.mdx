## 操作方法

执行此指令即可：

```plain
/premium
```

:::caution
* **非正版请勿操作**，会导致无法进入游戏！  
* 正版验证会**丢失**你正在游戏的数据！
* 如需迁移游戏数据，请联系管理员。

*🚧 自动迁移系统正在路上…*
:::

## 正版玩家权益

- 正版称号（自动发放，称号会显示在名字前面）
- 在线皮肤（由服务器显示皮肤和披风，且无需任何 Mod）
- 免密登陆（加入服务器不再需要密码登陆）
- 更多领地成员（由10个变为20个，您可以使用 `/lands edit <领地名>` 和 `/lands trust <玩家名>` 邀请其他玩家）
- 修改玩家名保留游戏数据（[minecraft.net](https://www.minecraft.net/en-us/msaprofile/mygames/editprofile) 修改玩家名后再次正版认证即可）
